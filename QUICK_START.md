# 🚀 Quick Start Guide

## Frontend (Next.js)

```bash
cd frontend
npm install
npm run dev
```

**Access:** http://localhost:3000

---

## Backend (Excel Import)

```bash
cd backend
./scripts/run_import.sh
```

**That's it!** The script handles everything automatically.
